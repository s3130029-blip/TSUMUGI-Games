# art/ — 元画像（マスター）置き場

ビルドには使わない**元データ（マスター）**を保管する場所です。
公開フォルダ `tamago-gacha/assets/` はデプロイ時に丸ごと置き換えられるため、
元画像をそこに置くと失われます。マスターはこの `app/art/` に置いて管理します。

- `scene_picture.png` … ガチャ画面の背景の元画像（1243×752・約1.3MB）。
  - 実際にアプリで読み込むのは、これを圧縮した `app/src/assets/scene_picture.jpg`
    （mozjpeg q85・約124KB）です。差し替え時はこのPNGを更新 → 再圧縮してください。
