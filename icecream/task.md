- [x] 背景をサーティワン風の店内画像に差し替え（`assets/store_image.png`、`body`／モード選択に敷く。前景UIは白を濃いめにして可読性確保）
- [x] 積む場所（`.stage`）に白いスポットライト状フィルターを追加し、積んだアイスを背景画像に埋もれず見えるように（中央〜下を明るく、外側は店内が薄く透ける）
- [x] 客のリアクション・コンボ・評価（要件 3.7）
      - 評価：自力ノーミス=ぴったり⭐⭐⭐／自力ミスあり=せいかい⭐⭐／こたえを見た=できた⭐（前向きな文言）
      - コンボ：自力クリア連続で加算しHUDに「🔥 コンボ ×N」。こたえを見たら0に（ミスでは切れない＝やさしめ）
      - 客リアクション：クリア時に顔が😍/😋に変わって弾み、ハート／星が浮かぶ（`Render.customerReact`）

- [x] 音（効果音／BGM）と ON/OFF 切り替え（`js/sound.js`）

      - iOS対策：初回タッチで AudioContext を解錠（`Sound.init` の unlock）
      - 🔊/🔇 ワンボタンで全音ON/OFF（右上 `#soundBtn`）。設定は localStorage に保存（失敗時は無視）
